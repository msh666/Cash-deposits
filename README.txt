Для того чтобы установить данное приложение, необходимо имень на компьютере установленное ПО Microsoft SQL Server.
Зайдя в него, восстановить базу данных Денежные_вклады.bak
После этого можно запускать непосредственно само приложение Денежные вклады.exe




Приложение реализовано на языке C# в среде Microsoft Visual Studio Ultimate 2013 v.12.0.31101.00 Update 4
.NET Framework v.4.5.52264
С помощью Microsoft SQL Server 2014 v.12.0.2254.0
